🎨 Picom UI Library - Pure JS Usage

This example shows how to use Picom UI components without any external dependencies.

📝 Input Component

// Create an input component const input = new InputComponent({ id: 'my-input', label: 'Your Name', placeholder: 'Enter your name...' }); // Render it to a container const container = document.getElementById('input-container'); container.appendChild(input.render());

🔘 Button Component

// Create a button component const button = new ButtonComponent({ id: 'my-button', text: 'Click Me!', onClick: () => alert('Hello from Pure JS!') }); // Render it to a container const container = document.getElementById('button-container'); container.appendChild(button.render());

☑️ Checkbox Component

// Create a checkbox component const checkbox = new CheckboxComponent({ id: 'my-checkbox', label: 'I agree to the terms', checked: false, onChange: (e) => console.log('Checked:', e.target.checked) }); // Render it to a container const container = document.getElementById('checkbox-container'); container.appendChild(checkbox.render());

📋 Select Component

// Create a select component const select = new SelectComponent({ id: 'my-select', label: 'Choose your favorite color', options: ['Red', 'Green', 'Blue', 'Yellow'], onChange: (e) => console.log('Selected:', e.target.value) }); // Render it to a container const container = document.getElementById('select-container'); container.appendChild(select.render());

🔧 Component Manager

// Get component statistics const stats = window.ComponentManager.getStats(); console.log('Total components:', stats.total); console.log('Memory usage:', stats.memoryUsage, 'bytes'); // Dispose all components window.ComponentManager.disposeAllComponents();
